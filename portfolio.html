Normalización y analisis de datos mediante SQL, Dashboard mediante Tableau

Mediante el uso de SQL se realizó un trabajo de normalización de un conjunto de datos de un punto de venta. Posteriormente se llevó a cabo un análisis exploratorio considerando distintos parámetros (edad, suscripción, formas de pago, condiciones de envío) y se definieron posibles KPIs relevantes para el área comercial, como porcentaje de clientes suscriptos, total de ventas y variación de ganancias según rating de productos. Finalmente, los resultados se plasmaron en Tableau para presentarlos de manera clara y visual.
